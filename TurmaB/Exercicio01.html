HTML básico - códigos HTML

Aprenda os conceitos básicos para a confecção de um site html para que você possa fazer seu primeiro site

Motivação


HTML (Abreviação de Hypertext Markup Language) é uma linguagem de marcação utilizada na estruturação de páginas web. Sua sintaxe é bastante simples e, assim como a XML é baseada em tags, que representam os diversos elementos de uma página, como imagem e links.

Ao acessar uma página web através de um navegador, ele é capaz de interpretar o código HTML e renderizá-lo de forma compreensível para o usuário final, exibindo textos, botões, etc. com as configurações definidas por meio das diversas tags que esta linguagem dispõe.
